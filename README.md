# Biscuit website

## Installing

The biscuit website uses the [Zola static site generator](https://www.getzola.org) to compile the
markdown content into the final website. Follow their instructions to install Zola on your system.

```sh
# From wc/ directory
npm install
```

## Quick start

```sh
# From wc/ directory
npm run build

# From root directory
zola serve
```

## Build

```sh
# From wc/ directory
npm run build

# From root directory
zola build
```
